# 注意事项

部分站点需要安装 `ignore-x-frame-headers` 扩展绕过 iframe 限制，否则脚本无法正常工作

安装扩展：
[Chrome][ignore-x-frame-options-chrome]
[Firefox][ignore-x-frame-options-firefox]
